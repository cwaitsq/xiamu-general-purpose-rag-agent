## 3. 启动顺序

1. 启动 Gateway Runner
   `python tools\启动_gateway_runner.py`
2. 启动后端
   `python tools\启动_backend.py`
3. 启动前端
   `cd frontend && pnpm dev --port 3001`
4. 打开 n8n
   `http://127.0.0.1:5678`
