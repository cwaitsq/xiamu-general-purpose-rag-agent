# 最小起订量与定制规则

## 最小起订量

- 常规产品按单个产品型号执行最小起订量
- 不同材质、不同尺寸、不同包装要求，最小起订量可能不同

## logo 定制

- 支持丝印、贴标、激光、吊牌等常见定制方式
- logo 定制通常需要客户提供可编辑源文件

## 包装定制

- 支持外箱、彩盒、标签、说明书等包装定制
- 包装定制通常有最小印刷量要求

## 图稿确认

- 客户确认图稿后再安排正式生产
- 图稿确认后如再次变更，可能影响交期和成本

## 大货确认

- 定制订单正式生产前，需要确认样品或最终图稿
- 未确认前，不建议直接安排大货

## 对外答复口径

系统可以回答标准起订量和定制规则。  
如果客户问的是超低 MOQ、特殊工艺、紧急定制，统一转人工处理。
