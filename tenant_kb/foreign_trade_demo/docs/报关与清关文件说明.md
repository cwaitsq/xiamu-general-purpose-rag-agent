# 报关与清关文件说明

## 常规可提供文件

- 商业发票
- 装箱单
- 提单或运单资料

## 按需确认文件

- 原产地证
- 熏蒸证明
- 检测报告
- 其他目的国要求文件

## 文件确认时间

- 特殊文件需在下单或出货前提前确认
- 临近出货再补提文件要求，可能影响交期

## 清关责任

- 不同贸易条款下，清关责任划分不同
- 客户需提前确认目的国进口要求

## 对外答复口径

系统可以回答常规清关文件范围。  
如果客户问的是目的国法规、清关失败责任、特殊认证要求，统一转人工处理。
