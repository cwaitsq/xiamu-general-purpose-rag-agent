# 转人工规则

## 必须转人工的场景

- 大客户议价
- 账期申请
- 账户变更
- 合同条款确认
- 清关责任争议
- 延期赔付争议
- 重大质量投诉
- 索赔金额确认

## 建议转人工的场景

- 知识库里没有明确答案
- 客户问题涉及多个异常同时出现
- 客户提供的信息不足，无法确认订单背景

## 系统答复原则

- 系统只能回答标准规则
- 没有证据时不能硬答
- 涉及价格、赔付、责任划分时优先转人工

## 对外答复口径

如果系统无法确认答案，统一使用保守答复，并建议客户联系人工客服继续处理。
