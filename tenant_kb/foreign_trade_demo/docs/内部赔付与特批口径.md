# 内部赔付与特批口径

## 适用场景

- 我方责任导致的大货延期
- 我方责任导致的明显批量质量问题
- 我方责任导致的单据错误影响出货

## 处理原则

- 先核实订单、责任、损失范围
- 未核实前，不对外承诺赔付金额
- 涉及赔付、补货、折让时，需内部审批

## 常见处理方式

- 补发货物
- 下单抵扣
- 折扣处理
- 其他经审批通过的方案

## 使用限制

- 本文档仅供内部参考
- 不能直接作为对外承诺
- 对外统一由人工客服或业务负责人确认
