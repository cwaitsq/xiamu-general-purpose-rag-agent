# 客户培训视频_转写样例

## 今天这段培训主要讲外贸报价、付款方式和交期确认。

客服在报价前必须先确认产品规格、采购数量、包装要求和贸易条款。
如果客户问报价是否含运费，不能直接回答包含，要先确认是 EXW、FOB、CIF 还是 DDP。
样品单一般先全款，大货订单通常先付定金，尾款节点按合同确认。
常规生产周期在样品确认和定金到账后开始计算，一般是十五到三十天。
如果客户要求加急，一定先找工厂确认排产，不能在客服侧直接承诺。
