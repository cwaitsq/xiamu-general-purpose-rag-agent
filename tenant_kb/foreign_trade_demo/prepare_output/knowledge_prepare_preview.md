# 外贸客服场景定义

## 1. 这个客服是干嘛的

这里的客服不是处理普通电商零售咨询，而是处理外贸业务里的客户沟通问题。

服务对象一般是：

- 海外采购商
- 海外经销商
- 海外品牌客户
- 长期合作客户

## 2. 主要回答什么问题

第一版先聚焦这些高频问题：

- 产品能不能定制
- 最小起订量是多少
- 能不能先寄样品
- 报价是不是含运费
- 支持什么付款方式
- 生产周期多久
- 发货方式有哪些
- 能提供哪些清关文件
- 有质量问题怎么处理

## 3. 哪些问题不能让系统自己硬答

这些问题优先转人工：

- 大客户议价
- 赔付争议
- 合同条款确认
- 账户变更
- 清关责任争议
- 特批折扣
- 大货延期争议

## 4. 第一版目标

第一版不是替代业务员，也不是替代跟单。

第一版目标很实际：

- 先回答标准问题
- 先减少重复沟通
- 先把资料查找效率提上来
- 遇到高风险问题就转人工
